# how-to-npm-jp

モジュールの作成方法を教えてくれるモジュールです。

## USAGE

```
npm i -g how-to-npm-jp
how-to-npm-jp
```

このパッケージは、作業環境の構築、依存パッケージのインストール、
npm へのログイン、モジュールの公開などなど、モジュールの基本を
体験させてくれます。また、すべての作業は自分のPC上で安全に実行
されます。
